### 1\. Prerequisites

NodeJS Version 14.x.x

yarn - `npm install -g yarn`

### 2\. Clone this repository

`git clone https://github.com/snyk/sweater-comb.git`

then `cd` into your local copy.

### 3\. Install

```
yarn
```

### 4\. Running tests

```
yarn test
```
